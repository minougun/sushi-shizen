# Sushi Zen UI改善作業サマリ（2026-04-03）

## 概要

HANDOFF.mdに基づき、インバウンド外国人観光客向け寿司店サイト（https://sushi-zen.vercel.app/）のUI評価と改善を実施。写真以外の全改善項目を対策し、情報量の大幅圧縮を行った。

### Phase 2: 残り4項目の対策

**コミット**: `6bc05fa`

| # | 改善 | 詳細 |
|---|------|------|
| 1 | モバイルヒーロー削減 | booking_panel + trust_marksをモバイル(767px以下)で非表示 |

#### KR翻訳の主な修正

- 맛집 사이트 → 레스토랑 예약 사이트（カジュアルすぎ→高級店に適切）
- 전자머니 → 전자화폐（日本語カルクを自然な韓国語に）
- d Pay / Rakuten Pay → d페이 / 라쿠텐페이（韓国語UI内のローカライズ）
- 서는 → 직접 맡는（日本語の直訳を自然な表現に）

#### ZH翻訳の主な修正

- 食べログ → Tabelog（日本語テキスト残りを全箇所除去）
- 米饭 → 醋饭（寿司飯の正しい中国語表現）
- 鱼料 → 食材（不自然な造語を一般的な語に）
- 门店 → 店铺（チェーン店風→個人店に適切）
- 手磨 → 现磨、来店客人 → 宾客（自然な中国語表現に）
